## Instalacja
Pobierz pliki za pomocą git clone 
```bash
git clone https://github.com/Sebastian-Nosal/Image-Processing.git
```

## Wymagania
python 3.7 lub nowszy

## Uruchomienie
```bash
python server.py
```

## Dokumentacja

- [github-pages](https://sebastian-nosal.github.io/Image-Processing/)

- lokalna: http://localhost:8000/docs/  po uruchomieniu aplikacji

## Funkcjonalności
- wczytywanie obrazu z pliku
- zamiana na skalę szarości (grayscale)
- obliczanie progu binaryzacji metodą Otsu
- ręczne dostosowywanie progu binaryzacji za pomocą suwaka
- binaryzacja dokumentu w oparciu o globalny próg
- nakładanie binaryzacji na oryginał lub na skalę szarości
